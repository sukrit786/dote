Grace WordPress Theme
---------------------

Created by sitechurch.com


Install
-------
1. Upload the grace folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Be sure to activate the Grace child theme, and not the Grace parent theme.
